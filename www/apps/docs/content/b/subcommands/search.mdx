# b search

The `b search` command helps you discover binaries available in the b CLI ecosystem, showing detailed information about each package including versions, descriptions, and installation commands.

## Synopsis

```bash
b search [options] [query]
```

## Description

The `search` command queries the b CLI package registry to find binaries matching your search criteria. It's the best way to discover new tools and check available versions before installation.

## Options

- `--output, -o`: Output format (json, yaml, table)
- `--help, -h`: Show help information

## Examples

### Basic Search

```bash
# Search for a specific binary
b search jq

# Search with partial match
b search py
```

### Advanced Search

```bash
# Show all available packages
b search

# Get results in JSON format
b search --output json jq
```
