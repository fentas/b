# b cache

Manage the local git cache used for env file syncing.

## Usage

```bash
b cache [subcommand] [flags]
```

## Subcommands

### b cache path

Print the cache directory path (typically `~/.cache/b/repos`).

```bash
b cache path
```

### b cache clean

Remove all cached git repositories and report freed disk space.

```bash
b cache clean
```
