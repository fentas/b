# b verify

Verify all installed binaries and env files against SHA256 checksums stored in `b.lock`. Reports missing files, checksum mismatches, or local changes.

## Usage

```bash
b verify [flags]
```

## Examples

### Verify all artifacts

```bash
b verify
```

On success the output ends with:

```
All artifacts verified âœ“
```

If any checksum mismatches are found, `b verify` exits with a non-zero status code and reports the count.

## Flags

| Flag         | Description               |
|--------------|---------------------------|
| `-h`, `--help` | help for verify           |

## Global Flags

| Flag                 | Description                                                              |
|----------------------|--------------------------------------------------------------------------|
| `-c`, `--config string`  | Path to configuration file (current: `/home/fentas/github/fentas/b/.bin/b.yaml`) |
| `--force`            | Force operations, overwriting existing binaries                          |
| `-o`, `--output stringArray` | output options: json\|yaml\|format                                  |
| `-q`, `--quiet`      | Quiet mode                                                               |
| `-v`, `--version`    | Print version information and quit                                       |
